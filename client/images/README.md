# 图片资源说明

此目录用于存放小程序的图片资源，包括：

## TabBar 图标

需要准备以下图标文件（建议尺寸：81px × 81px）：

- `home.png` - 首页图标（未选中）
- `home-active.png` - 首页图标（选中）
- `create.png` - 发布图标（未选中）
- `create-active.png` - 发布图标（选中）
- `profile.png` - 个人中心图标（未选中）
- `profile-active.png` - 个人中心图标（选中）

## 其他图片

- `default-avatar.png` - 默认头像（建议尺寸：200px × 200px）

## 图标设计建议

- 使用简洁的线条图标
- 选中状态使用主题色（#4CAF50）
- 未选中状态使用灰色（#7A7E83）
- 保持图标风格统一

## 在线图标资源

可以使用以下网站获取免费图标：
- [Iconfont](https://www.iconfont.cn/)
- [Flaticon](https://www.flaticon.com/)
- [Icons8](https://icons8.com/)
